# 动态表单
基于bootstrap实现的，可支持拖拉拽的动态表单生成工具<br/>
####1、可从左侧随意拖拽
####2、可以设置栏位
####3、可对拖拽后的属性进行设置
####4、可设置单列、双列、删除列

前期功能很简单 
持续更新中<br/>
